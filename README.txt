Riego Plantas PWA

Incluye:
- Carga de plantas con foto
- Tipo sugerido/manual
- Frecuencia de riego
- Último y próximo riego
- Botón "Regada hoy"
- Historial interno
- Backup exportar/importar
- Funciona offline cuando se publica como web/PWA

Para usar en iPhone:
1) Publicar la carpeta en Netlify Drop o GitHub Pages.
2) Abrir la URL en Safari.
3) Compartir > Agregar a pantalla de inicio.

Nota: la identificación automática real por foto requiere una API externa. Esta versión permite guardar foto y clasificar manualmente con opciones frecuentes.
